# Clavis 🔑

> Application de gestion de clés pour promoteur immobilier.

Permet de tracer en temps réel à qui chaque clé d'un parc immobilier est attribuée, avec scan QR code, signature digitale du receveur, et historique complet des mouvements.

## Stack

- Next.js 15 (App Router) + TypeScript
- Tailwind CSS + shadcn/ui
- Supabase (Postgres + Auth + Storage)
- Vercel (déploiement)
